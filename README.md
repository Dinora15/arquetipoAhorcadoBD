# arquetipoAhorcadoBD
 Juego ahorcado base de datos
 
 Versión Funcional del juego ahorcado escrito en Java, con almacenamiento de 
información  a una Base de Datos mysql.
Estructura Basada en el Diagra de Clases UML y Caso de Uso.
